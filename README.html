Installation for Windows: (as part of this repository on github)

Please report any issues on the main repository!!!

Windows with g++

Run the 'ihavegcc.bat' file from the windows folder in the extracted main folder.
A file named 'calc.exe' will appear in the src folder. Your compiled calculator is there! Move it as you wish!

Windows with Python

Run the 'ihavepython.bat' file from the windows folder in the extracted main folder.
It will run directly for you, but you can't move this file. It is the only ay to run other than downloading the binary or having g++.

Installation for Linux:

Linux with g++

In the extracted folder, in linux, open your terminal there.
Type bash ihavelinuxgcc.sh, then press enter. There will be a binary file in the src folder.
To run it again, open up the terminal in whatever folder you put the binary file (named calc) in, and type ./calc

Installation for linux without g++

In the extracted folder, go to linux, then open the terminal there.
In the terminal, type bash ihavelinux.sh. It will run the calc.

You cannot move this file, unless you cange the cd in the file to go to whatever directory you have the .py file in.
